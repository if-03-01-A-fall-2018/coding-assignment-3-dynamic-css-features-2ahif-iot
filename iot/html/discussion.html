Discussion

Do we really need the Internet of Things?

That's not an easy question! Smart devices are a very useful thing.
We can't stop the future: The Internet of Things will be a huge topic in the next few years.

Smart devices in new and old buildings

It's easier to integrate IoT devices into new buildings if they require a wired connection.
Because of short-range wireless technologies like WiFi or ZigBee, old buildings can use the advantages too.

Problems of wireless connections

The major issue with wireless connections is definitely the range.
This issue can be eliminated by using range extenders or repeaters for the needed technology.

Required skills

Some things might also not be that easy to set up or even require coding skills.

Creating selfmade projects using development boards

With development boards like Arduinos, NodeMCUs (including the ESP8266 WiFi adapter) or more sophisticated devices like the Raspberry Pi, people can build own projects for the IoT and more.
Click on one of the images to get more information about the specific board.

NodeMCU

The NodeMCU is based on a ESP8266 Wi-Fi chip and is often being programmed using the Lua scripting language.
Thanks to the onboard Wi-Fi it's suitable for many smart home and IoT applications.

Arduino

There are many Arduino single-board microcontrollers, the most popular one is definitely the Arduino UNO.
The programming language used is C, the most used IDE is the Arduino IDE.

Don't know where to start? Here's a list of project ideas!

Raspberry Pi

The Raspberry Pi is a single-board computer, the operating system is stored on an SD card.
It was initially developed for schools to promote basic computer science in development countries.

There's almost no project that can't be done thanks to the USB ports and the GPIO pins.
