Dash Button

Demonstration

We'll buy a Dash Button and show how it works.

The intended use for an Amazon Dash Button is to buy a product just by tapping on the button.
This can't be done with magic. It has to connect to the internet using a WiFi network.
After that, it automatically orders the selected product.

"Hacking" it

People across the internet found ways to use the Dash Button for their own projects.
The button replies to ping/ICMP requests and gets an IP address via DHCP.

Pinging the button every X seconds is a nice way to detect a button press.
A Raspberry Pi or another server could then execute a specified task, e.g. turning the lights on.

After integrating it into a running system, disallowing internet access wouldn't be a bad idea.
Why? Amazon doesn't have to trace every button press.
