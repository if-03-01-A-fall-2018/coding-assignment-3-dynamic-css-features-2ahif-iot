Heating stats: Description

Backend description for the "Heating stats" page.

The FHEM home automation server queries the data from the heating system via an RS232 serial server running on a Raspberry Pi.
FHEM also generates the plots and uploads the queried data to an external web server.

The reverse engineered protocol used by the control unit is described here.
It consists of several actions (read, write, set time/date) and uses the internal memory addresses of the microcomputer.
